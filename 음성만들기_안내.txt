한글 놀이터 - 새 성우(ElevenLabs, 무료) 음성 넣기 안내
======================================================

[준비물] 컴퓨터(노트북), 파이썬, ElevenLabs 계정 (무료, 신용카드 불필요)

1) ElevenLabs 가입 + API 키
   - https://elevenlabs.io 가입(무료)
   - 우측 상단 프로필 -> 'API Keys' -> 새 키 생성 -> 키 복사
   - 무료 플랜: 월 10,000자. 우리 앱 전체는 약 5,000자라 한 달 안에 다 만들 수 있어요.
   - (무료 결과물은 비상업적/출처표기 조건. 가정용은 괜찮아요.)

2) (권장) 한국어 보이스 고르기  ※ 발음이 훨씬 자연스러워져요
   - ElevenLabs 사이트의 'Voices' 또는 'Voice Library'에서 'Korean' 검색
   - 마음에 드는 여성/아이(밝은)/남성 보이스를 내 보이스에 추가
   - 각 보이스의 ID 복사 방법: generate_voices.py 에서 LIST_VOICES = True 로 바꾸고
     한 번 실행하면 내 보이스 이름과 ID가 출력됩니다.
   - 출력된 ID를 SPEAKERS 의 f / kid / m 에 붙여넣고, 다시 LIST_VOICES = False 로!

3) 파이썬 준비
   - 파이썬 설치(python.org). 설치 시 "Add to PATH" 체크
   - 터미널/명령프롬프트:  pip install requests

4) 스크립트에 키 넣고 실행
   - generate_voices.py 를 메모장으로 열어 API_KEY 에 1)의 키 붙여넣기
   - (2)에서 정한 voice ID도 SPEAKERS 에 넣기
   - 실행:  python generate_voices.py
   - audio/f, audio/kid, audio/m 폴더에 mp3 생성 (이어하기 됨)

5) 깃허브에 올리기  ※ 파일이 4천 개가 넘어 웹 업로드는 비효율적
   - 권장: 'GitHub Desktop' 설치 -> 저장소 열기 -> audio 폴더 통째로 넣기 ->
           Commit -> Push (한 번에 업로드)
   - audio 폴더는 index.html 과 같은 위치(저장소 맨 위)에 있어야 합니다.

6) 아이패드에서 적용
   - 앱 완전 종료 후 다시 열기 -> 설정 ⚙️ -> 성우(목소리)에서 여성/아이/남성 선택

* 소리 파일이 아직 없으면 앱은 기존처럼 '기기 음성'으로 말합니다(고장 아님).
* 받침/글자 만들기까지 전부 새 성우로 나옵니다.
* 한국어 발음은 (2)에서 네이티브 한국어 보이스를 고를수록 좋아집니다.
