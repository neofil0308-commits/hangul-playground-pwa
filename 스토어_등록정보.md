# Play 스토어 등록정보 문안

> Play Console의 «기본 스토어 등록정보»에 그대로 복붙하기 위한 문안.
> 글자 수 검증: `python tools/check_store_listing.py` (제한 초과 시 종료 코드 1)
>
> 원칙: **과장하지 않는다.** Play는 오해를 부르는 설명을 정책 위반으로 본다.
> 특히 ① TWA라 첫 실행에 인터넷이 필요하다는 점 ② 학습 성과를 보장하지 않는다는 점을 숨기지 않는다.

---

## 앱 이름 (최대 30자)

```
하니의 한글 모험 - 유아 한글 떼기
```

> 「하니의 한글 모험」만 쓰면 8자로 짧고 깔끔하지만, 스토어 검색에서 '유아 한글'로 찾는
> 부모에게 걸리지 않는다. 뒤에 붙인 설명구는 키워드 나열이 아니라 실제 내용이라 정책상 문제없다.

---

## 짧은 설명 (최대 80자)

```
글자를 모르는 네 살도 그림과 소리로 시작해요. 8막 95화를 끝내면 편지를 혼자 읽어요.
```

> 스토어 목록에서 앱 이름 바로 아래 보이는 한 줄. 대상(네 살·백지 상태)과
> 끝났을 때 무엇이 되는지(편지를 혼자 읽음)를 한 문장에 담았다.

---

## 자세한 설명 (최대 4000자)

```
한글을 하나도 모르는 아이가, 공부가 아니라 모험을 하다가 글자를 떼는 앱입니다.

■ 이야기를 따라가면 그게 곧 한글 순서예요

어느 날 별빛 우체국의 편지에서 글자가 모두 사라졌어요. 병아리 하니와 함께 사라진 글자를 하나씩 되찾는 여정이 시작됩니다.

이 모험의 순서가 곧 한글을 배우는 순서입니다. 모음 → 자음 → 글자 합치기 → 받침 → 된소리 → 복모음 → 단어 → 문장. 마지막 화에서 아이는 맨 처음 하얗게 지워졌던 편지를 스스로 읽게 됩니다.

■ 8막 95화의 여정

1막 모음의 빛 (글자 숲) — 기본 모음 ㅏㅓㅗㅜㅡㅣㅑㅕㅛㅠ
2막 자음 친구들 (글자 숲) — 기본 자음 ㅁㄴㅇㄱㄷㅂㅅㄹㅈㅊㅋㅌㅍㅎ
3막 글자 공방 — 자음과 모음을 합쳐 가·나·다 만들기
4막 받침의 문 — 받침 ㄱㄴㄷㄹㅁㅂㅇ
5막 쌍둥이 소리 (소리 동굴) — 된소리 ㄲㄸㅃㅆㅉ
6막 숨은 모음 — 복모음 ㅐㅔㅚㅟㅘㅝ 등
7막 단어 마을 (단어 동산) — 쉬운 것부터 단어 읽기
8막 이야기 책 (별빛 우체국) — 문장 읽기, 그리고 졸업

글자 하나가 한 화입니다. 하루에 한 화씩이면 약 13주 분량이에요.

■ 글을 못 읽는 아이를 위해 만들었습니다

- 설명을 글로 하지 않습니다. 큰 그림과 하니의 목소리가 안내해요.
- 모든 글자와 낱말에 사람이 읽는 듯한 음성이 붙어 있습니다. 기계음이 아니에요.
- 모양만 외우게 두지 않고 "이응에 ㅗ를 더하면 오"처럼 원리를 알려줍니다.
- 판정이 관대합니다. 만나고, 짝 맞추고, 소리로 찾으면 익힌 것으로 봐요. 좌절할 일이 없습니다.
- 작은 성취는 작게, 큰 성취는 크게 칭찬합니다.

■ 하루 흐름

홈에서 오늘의 모험 세 가지를 보여줍니다. 글자 숲에서 오늘의 글자를 만나고, 단어 동산에서 자음과 모음을 끌어다 붙여 보고, 소리 동굴에서 소리만 듣고 글자를 찾아요. 마지막에 오늘 배운 것을 정리하고 스티커를 받습니다.

지난 글자는 잊을 때쯤 다시 나옵니다. 아이마다 맞히는 정도가 달라서, 어려워한 글자가 더 자주 돌아와요.

■ 부모님을 위해

- 아이가 몇 글자를 익혔고 무엇을 어려워하는지 볼 수 있는 학습 기록이 있습니다.
- 설정으로 들어가려면 간단한 숫자 문제를 풀어야 해요. 아이가 혼자 설정을 바꾸지 못합니다.
- 아이가 여럿이면 프로필을 나눠서 각자 진도를 따로 쌓을 수 있습니다.
- 목소리, 말하기 속도, 음량, 배경음악을 아이에게 맞게 조절할 수 있어요.

■ 광고도, 결제도, 회원가입도 없습니다

- 전체 기능이 무료입니다. 인앱결제나 구독이 없어요.
- 광고를 넣지 않았습니다. 아이가 실수로 광고를 누를 일이 없습니다.
- 회원가입과 로그인이 없습니다.
- 학습 기록은 기기 안에만 저장되고 어디로도 전송되지 않습니다. 저희도 볼 수 없어요.
- 광고 SDK와 분석 도구를 일절 쓰지 않고, 글꼴과 음성까지 앱에 함께 담아 외부 서버를 부르지 않습니다.

■ 알아두실 것

- 처음 실행할 때는 인터넷 연결이 필요합니다. 한 번 받아두면 그 다음부터는 인터넷 없이도 됩니다.
- 가로로 넓은 화면에 맞춰 만들었습니다. 태블릿을 가로로 놓고 쓰시는 것을 권합니다.
- 학습 속도는 아이마다 다릅니다. 이 앱은 한글 떼기를 돕는 도구이고, 정해진 기간 안의 성과를 약속하지는 않습니다.

■ 만든 이유

네 살 아이에게 한글을 가르쳐 보려다, 글을 못 읽는 아이에게 글로 설명하는 앱들뿐이라는 걸 알았습니다. 그림과 소리만으로 끝까지 갈 수 있는 여정을 만들고 싶었습니다.

웹에서도 그대로 쓰실 수 있습니다.
https://neofil0308-commits.github.io/hangul-playground-pwa/
```

## 사실 확인 근거

문안에 쓴 숫자는 전부 코드에서 뽑았다(추정치 아님).

| 주장 | 근거 |
|---|---|
| 8막 95화 | `app-data.js`의 `EPISODE_PATH` 길이 = 95, `CURRICULUM` 길이 = 8 |
| 막별 화 수 | 10·14·14·14·5·11·12·15 (합 95) |
| 음성이 기계음이 아님 | edge-tts 신경망 음성 3종(`generate_voices.py`) · 음성 파일 4,989개 |
| 외부 서버 호출 없음 | 폰트 self-host(`assets/fonts/`), 광고·분석 SDK 없음 |
| 오프라인 동작 | `sw.js` 서비스 워커 precache |
| 첫 실행 인터넷 필요 | TWA는 웹을 띄우는 껍데기 — AAB에 콘텐츠가 들어있지 않다 |
