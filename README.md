# 한글 놀이터

아이들이 자음·모음·단어·문장·소리 놀이를 통해 한글을 익히는 정적 PWA입니다.

## 현재 구조

- `index.html` — 앱의 HTML/CSS/JavaScript 대부분이 들어있는 단일 파일
- `sw.js` — 오프라인 캐시용 service worker
- `manifest.json` — PWA 설치 설정
- `audio/` — 한글 음성 MP3 리소스
- `icon-192.png`, `icon-512.png`, `apple-touch-icon.png` — PWA 아이콘
- `generate_voices.py` — 음성 리소스 생성 보조 스크립트

## 로컬 실행

Git Bash 또는 터미널에서 프로젝트 폴더로 이동한 뒤 실행합니다.

```bash
python -m http.server 5187 --bind 127.0.0.1
```

브라우저에서 엽니다.

```text
http://127.0.0.1:5187/index.html
```

## 현재 기능

- 오늘의 미션
- 글자 친구들
- 글자 만들기
- 단어 공부
- 문장 쓰기
- 따라쓰기
- 짝 맞추기
- 소리 퀴즈
- 듣고 찾기
- 스티커 보상
- 설정
- PWA 설치/오프라인 캐시

## 고도화 방향

단순 기능 모음에서 벗어나, 캐릭터가 이끄는 스토리 기반 학습 경험으로 확장합니다.

1차 방향:

- 병아리 마스코트를 `하니` 캐릭터로 설정
- `오늘의 미션`을 `오늘의 모험`으로 재구성
- 기존 기능을 스토리 퀘스트 흐름으로 연결
- 미션 완료 시 캐릭터 대사와 보상감을 강화
- 현재 기능 로직은 최대한 유지하고 홈/진행 경험부터 개선

## 작업 원칙

- 큰 변경 전 git 커밋으로 기준선을 남깁니다.
- 기능 추가보다 먼저 기존 앱이 깨지지 않는지 smoke test를 확인합니다.
- 대규모 구조 분리는 Story Adventure Shell 1차 적용 이후 진행합니다.
